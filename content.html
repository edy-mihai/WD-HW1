Project Portfolio

Van Westendorp model app

This project focused on data analysis and user pricing psychology. I built an application using Python that calculates the Van Westendorp Price Sensitivity Meter, allowing users to input survey data to determine the optimal price point for a new product and then generating a graph for easy understaning. This challenge helped me strengthen my understanding of backend logic, data processing, and creating clean user interfaces for complex mathematical models.

Flappy Bird in Unity

To dive deep into the fundamentals of game physics and state management, I built a fully functional clone of the classic game Flappy Bird. Using Unity and C#, I programmed the gravity mechanics, infinite pipe generation, and strict collision detection. This project was a massive stepping stone in my goal of joining a game development team, teaching me how to manage game loops and interactive UI scoring systems.

Unity tutorial

Based on what I learned while building the Flappy Bird project, I created an interactive Unity tutorial website designed to help other beginners grasp the basics of the engine. It walks users through navigating the interface, manipulating objects, and writing their first C# movement scripts. Building an educational tool actually solidified my own understanding of Unity's core architecture, physics materials, and camera mechanics.

The Volunteer app

I wanted to build software that makes a real-world impact, so I developed an app designed to connect local volunteers with organizations that need help. The app allows users to browse upcoming community events, sign up for shifts, and track their volunteer hours. This project pushed me to think carefully about database structures, user flows, and creating an intuitive, accessible front-end experience.
